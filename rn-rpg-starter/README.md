# React Native Expo + Firebase RPG Starter

A starter repo for an Expo (React Native) app with Firebase Auth + Firestore and RPG scaffold.

## Quickstart
1. `npm install`
2. Add Firebase config to `src/services/firebase.js`
3. `npx expo start`

### Scripts
- `npm start` → Expo dev server
- `npm run android` → Android
- `npm run ios` → iOS
- `npm run web` → web preview
